# Old MH Database

Database koleksi data dari beberapa game **Monster Hunter klasik**, digunakan untuk keperluan API, bot, atau tools komunitas.

Repository ini berisi data quest yang diambil dari berbagai sumber dan disimpan dalam format database agar mudah digunakan oleh aplikasi lain.

## Supported Games

Saat ini database mencakup:

- Monster Hunter Portable 3rd (MHP3rd)
- Monster Hunter Freedom Unite (MHFU)

Dan masih akan bertambah seiring waktu.

Struktur database dipisahkan berdasarkan game agar lebih mudah dikelola.
